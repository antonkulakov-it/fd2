Циклом

            var arr1=[1,2,3,4,5,6,7];
            
            for (var i=0; i < arr1.length; i++) {
                console.log(i,arr1[i]);
            }

С помощью методов массива

            arr1.forEach(function(p,i,arr){
                console.log(p,i);
            });
Методы map и filter также можно использовать для перебора элементов, хотя у них и другие задачи. 
            arr1.map(function(p,i,arr){
                console.log(p,i);
            });
            arr1.filter(function(p,i,arr){
                console.log(p,i);
            });
Методы map и filter возвращают новый массив! (можно использовать для копирования)

Конструкция for...in

            for (var i in arr1) {
                console.log(i, arr1[i]);
            }
Конструкцию for...in удобно использовать для перебора "дырявых" массивов. 
            var arr2 = [];
            arr2[1]=1;
            arr2[5]=2;
            arr2[9]=3;
            for (var i in arr2) {
                console.log(i, arr2[i]);
            }
