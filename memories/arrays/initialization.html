Объявление

Объявление массива - это указание, что некая переменная является массивом. 
            var arr=[];
            var arr=new Array();

Инициализация

Инициализация массива - это назначение конкретных значений данной переменной.

Способы инициализации массивов

При объявлении: 
            var arr1 = new Array(3,4,5);
            var arr2 = [3,4,5];
После объявления: 
            var arr3 = [];
            arr3.push(3,4,5);
            var arr4 = [];
            arr4.unshift(3,4,5);
            var arr5 = [];
            arr5.unshift(5);
            arr5.unshift(4);
            arr5.unshift(3);
