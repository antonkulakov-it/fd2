Что нужно, чтобы настроить gulp.

создаем папку проекта (все команды, набираемые в терминале будут обозначаться как "#$ команда"):
#$ mkdir projectName

переходим в папку проекта:
#$ cd projectName

инициализируем проект:
#$ npm init
Вводим ответы на вопросы.
Смотрим на появившийся файл package.json - это файл манифеста - описания проекта. Его можно редактировать как обычный текстовый файл 


Устанавливаем gulp (в зависимости от настроек системы нужно или одно или другое, но иногда и то и другое):
#$ npm install -g gulp
#$ npm install gulp

Устанавливаем необходимые модули:
#$ npm install browser-sync
#$ npm install gulp-rigger

Создаем gulpfile.js - файл конфигурации gulp:
#$ touch gulpfile.js
Что к чему в файле - см. комментарии к файлу. 


создаем папки проекта:
#$ mkdir app
#$ mkdir -p app/css
#$ mkdir -p app/js

создаем дефолтную страницу: 
#$ touch app/index.html

создаем папку дистрибутива:
#$ mkdir dist

Жестких требований по компоновке проектов не существует.
